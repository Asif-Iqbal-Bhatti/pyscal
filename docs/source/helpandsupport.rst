Help and support
================

In case of bugs and feature improvements, you are welcome to create a new issue on the `github repo <https://github.com/srmnitc/pyscal>`_. You are also welcome to fix a bug or implement a feature. Please see the `extending and contributing <https://pyscal.readthedocs.io/en/latest/extending.html>`_ section for more details. 

Any other questions or suggestions are welcome, please contact `us <mailto:sarath.menon@rub.de>`_.
